# Magic UI / Aceternity slot

Landing-page animated components live here (installed via `npx shadcn@latest add
"https://magicui.design/r/<component>"` — Magic UI ships shadcn-registry
compatible components; Aceternity components are copied in and adapted).

Rules (see CLAUDE.md → UI / Design Libraries):
- Adapt to our tokens (accent/panel/line/ink) — never ship default gradients.
- Landing page only; the dashboard stays shadcn-clean.
- Subtle motion, reduced-motion safe, no marquee overuse.
